F(oire) A(ux) Q(uestions)

Les encheres sont parfois différentes pour la même donne:

La méthode utilisée est statistique. En conséquence, les enchères peuvent être différentes quand les conventions sont différentes même si ces différences sont sans rapport avec la séquence d'enchère. Par contre, quand les conventions sont strictement identiques, les enchères le sont également.

Le Fichier d'aide ne fonctionne pas:

Le fichier d'aide Wbridge5.chm doit se trouver dans le même répertoire que l'exécutable Wbridge5.exe. Cliquer sur Wbridge5.chm. Si vous avez le message 'L'éditeur n'a pas pu être vérifié. Voulez-vous vraiment exécuter ce logiciel ?', décocher la case 'toujours demander ....' et cliquer sur Exécuter. Si le problème persiste (sur Vista), cliquer avec le bouton droit sur wbridge5.chm, cliquer avec le bouton gauche sur propriétes puis sur l'onglet sécurité et donner l'autorisation de lecture et d'écriture à tous les utilisateurs.

Peut-on jouer en ligne avec WBridge5 ?

Wbridge5 ne peut pas servir d’interface à un service de bridge en ligne comme CANALBRIDGE ou BBO. Par contre, il permet de jouer en réseau avec son partenaire en utilisant Bridge Moniteur comme serveur.

Il permet également de participer à un tournoi en ligne quotidien avec Bridgez.

J'ai installé WBridge5 et l'écran est illisible :

Dans le cas ou vous avez choisi la définition 800*600, Wbridge5 occupe tout l'écran et il est nécessaire de masquer la barre des taches de Windows.

En configuration grandes polices, les caractères sont tronqués. Il est nécessaire de mettre Windows dans sa configuration par défaut (96pp) pour avoir un affichage correct.

En fermant le programme, celui-ci me demande à chaque fois s'il faut enregistrer les préférences alors que rien n'a changé:

Lancer Wbridge5 puis supprimer Wbridge5.ini qui se trouve dans le même répertoire que Wbridge5.exe.

Fermer ensuite le programme et répondre oui quand il demande s'il faut enregistrer les préférences .

Ou peut-on trouver les fichiers PBN des grands tournois ?

 Sur le site de  Tis Veugen , où l'on trouvera également beaucoup d'autres choses concernant les  fichiers PBN et les programmes de bridge, et surtout sur le site de Nikos Sarantakos  . Sur ce site, beaucoup des fichiers sont au format lin qui est le format de BBO. Wbridge5 permet maintenant de transformer les fichiers Lin au format PBN (Menu Fichier). On peut enfin télécharger sur le site de BBO tous les tournois qui ont été transmis en ligne et les transformer au format PBN afin de les visualiser.

Comment générer plusieurs donnes avec les mêmes spécifications ?

Le programme garde en mémoire les dernières spécifications du générateur de donne. On peut donc générer une nouvelle donne avec les mêmes contraintes, en cliquant sur la petite flèche qui apparaît sur la barre supérieure d'outils ou en appuyant sur la touche F5. 

Qu'est-ce que le mode ?

Dans le cas de mode 1, le programme jouera de la même manière un contrat, quel que soit le nombre de plis demandés (style T.P.P). Il adoptera une ligne de jeu qui lui donnera en moyenne le maximum de plis.

Dans le cas de mode 0 (conseillé), il cherche à assurer le contrat ou la chute de celui-ci (style tournois par équipes). Ceci n'affecte que le jeu de la carte.

Comment échanger des données avec d’autres programmes?

Le Format standard des fichiers comportant des parties de bridge est le format PBN. Wbridge5 peut générer et lire des fichiers sous ce format. C’est également le cas pour la plupart des programmes de bridge (Jack, GIB, Q+, Sabrina, Bridge Baron, Blue Chip Bridge,GotoBridge).

Si on recommence  la même partie, le résultat précédent reste affiché.

C'est volontaire. Cela permet d'effectuer des comparaisons après avoir rejouer la même partie.

Comment imposer un contrat ?

Au niveau des enchères, il faut passer en mode manuel (Menu Jeu). Si l'on est intéressé uniquement par le résultat d'une donne à cartes ouvertes (le programme triche), Il suffit de faire saisir par le déclarant le contrat final. Dans le cas où le programme ne tricherait pas, il est par contre préférable de rentrer une séquence d'enchères vraisemblable car le programme tient alors compte des renseignements déduits des enchères.

Ou trouver des informations sur l'évaluation des mains ?

La méthode de Kaplan et Rubens est détaillée sur le site de   Jeff Goldsmith (en anglais).
Une comparaison entre les différentes méthodes d’évaluation faite à partir de simulations, se trouve sur le site de Thomas Andrew.

Comment demander le reste ?
 
Il suffit de demander au programme de jouer à sa place en cliquant sur Wbr5. On peut également cliquer sur tabler dans le menu Actions.

Toutes les enchères légales ne sont pas toujours accessibles.

La boîte d'enchère est déroulante. En utilisant les flèches du pavé de direction, il est possible d'accéder à toutes les enchères.

Existe-t-il une version pour Mac ?

Non, mais Wbridge5 peut fonctionner sur Mac avec un émulateur comme Virtual PC. ou Bootcamp en ce qui concerne les Macintel..

Existe-t-il une version pour Linux ?

Non, mais Wbridge5 peut fonctionner sous Linux avec un émulateur comme Cegeda ou Wine.
